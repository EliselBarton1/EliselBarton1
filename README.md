### What I do

I work at the intersection of **data integration, data quality, and business reporting**: building and monitoring ETL pipelines across HRIS, CRM, ticketing, and BI systems, writing the SQL checks that catch bad data before it reaches a dashboard, and documenting workflows so a fix outlives the ticket it came from.

- 🔗 **Data Integration** — Salesforce, Jira, Google Workspace, and HRIS systems synced through Integrate.io/MySQL pipelines
- ✅ **Data Quality & Monitoring** — null/duplicate checks, source-to-target row counts, freshness/lag alerting
- 📊 **BI & Reporting** — Power BI, Looker, KPI definitions tied to real business questions
- 🐍 **Automation** — Python scripts for cleanup, validation, and reporting so manual steps stop being manual
- 📝 **Documentation** — Jira-style write-ups (summary, repro, acceptance criteria, resolution) so process knowledge doesn't live only in someone's head

Currently doing this at JerseySTEM.
